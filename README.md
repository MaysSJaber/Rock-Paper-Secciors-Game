A robust, console-based implementation of the classic Rock, Paper, Scissors game. This project demonstrates core C++ programming concepts through an interactive and clean user interface.

🚀 Key Features
Dynamic Rounds: Play multiple rounds (1 to 10) in a single session.

Smart AI: Randomly generated computer moves using seeded randomization.

Real-time Visual Feedback: Screen colors change based on the round result (Green for Win, Red for Loss, Yellow for Draw).

Structured Data: Uses Structs and Enums to manage game state and choices efficiently.

Input Validation: Robust handling of user inputs to prevent game crashes.

Final Statistics: Comprehensive game summary showing wins, losses, and draws.

🛠️ Technical Concepts Covered
The project serves as a practical application for:

Control Flow: Using switch-case and do-while loops for game logic.

Functional Programming: Modular design with reusable functions.

System Calls: Using system("cls") and system("color") for UI enhancements.

Logic: Implementing the game rules using efficient conditional logic.
